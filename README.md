# CTPSharp
C#版CTP交易和行情接口类库，在原基础上进行二次封装，支持32位和64位自适应，支持多账户多实例，入参和出参全部使用C#实体类，返回结果支持MVVM绑定。
x86和amd64文件夹内的文件为MdApi.dll、TradeApi.dll、thostmduserapi.dll、thosttraderapi.dll，需要将这两个文件夹复制到当前程序文件夹。

CTP版本6.3.16_20190305
